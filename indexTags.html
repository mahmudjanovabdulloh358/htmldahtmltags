html - Hujjatning asosiy ildiz tegi.

head - Sahifa haqida meta ma’lumotlarni o‘z ichiga oladi.

title - Brauzer yorlig‘ida ko‘rinadigan sahifa sarlavhasi.

meta - Qidiruv tizimlari va brauzerlar uchun meta ma’lumotlar qo‘shadi.

link - CSS fayllar va boshqa resurslar bilan bog‘lanish uchun ishlatiladi.

style - HTML hujjat ichida CSS kod yozish uchun.

script - JavaScript kodlarni ulash yoki kiritish uchun.

body - Brauzerda ko‘rinadigan barcha kontent shu yerga yoziladi.

header - Sahifaning yoki bo‘limning sarlavhali qismi.

footer - Sahifaning yoki bo‘limning pastki qismi.

nav - Sayt navigatsiyasi (menyular va havolalar uchun).

main - Sahifaning asosiy mazmuni shu yerga yoziladi.

section - Katta bo‘limlarni ajratish uchun ishlatiladi.

article - Mustaqil kontent (blog posti, maqola, yangiliklar).

aside - Yon panel yoki reklama bloklari uchun.

div - Blok elementlarini ajratish uchun universal konteyner.

span - Matn yoki inline elementlarni ajratish uchun.

address - Muallif yoki sayt egasining aloqa ma’lumotlari.

h1/h6 - Sarlavha darajalari (h1 eng katta, h6 eng kichik).

p - Paragraf yaratish uchun ishlatiladi.

ul - Tartibsiz (belgili) ro‘yxat yaratish.

ol - Tartibli (raqamlangan) ro‘yxat yaratish.

li -Ro‘yxat ichidagi element.

dl - Tavsif ro‘yxati (Dictionary List).

dt - Tavsif qilinayotgan element (Definition Term).

dd - Tavsif (Definition Description).
